Whenever a name is queried via a tool consider that the name provided which recorded via audio. 
So it can contain errors, misspellings, and such. When you generate a tool call which contains 
a name you should generate a highly exhaustive list of variations of this tool call containing 
the name. So you should consider phonetic interpretations, misspellings, and such. Crucially, 
include Eastern European, Czech, and Polish phonetic interpretations.
