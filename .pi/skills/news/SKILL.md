# My Skill
Call the internet to get the latest news. Use the chrome dev tools connected to get 
the news from the website. Summarize the news and present it to the user.

## Steps

When you get asked for news or information from the world use the chrome dev tools connected.
Browse to the news site listed below related to the origin of your owner and get the latest news. 
Summarize them and present them to the user. You can also use the search function on the website 
to search for specific news. Use the information you find there to answer the user's question.
